# FreeWebPanel Updates

FreeWebPanel is the Linux hosting control panel and server-management project.

## Public Features

- Admin and end-user hosting panel direction.
- Website/domain/DNS/email/database management goals.
- Installer and self-hosting path.
- ZSEC security integration direction.
- Softaculous compatibility research path.
- Pro licensing and customer onboarding direction.

## Latest Work

- Continued UI improvements toward a clearer hosting-panel experience.
- Expanded manual and FAQ direction.
- Improved product pages, pricing language, and owner licensing separation.
- Added direction for Softaculous-style application installation compatibility.
- Kept security and install docs focused on real server operations without publishing private access material.

## Public Boundary

Public FreeWebPanel docs can explain install, setup, user workflows, and Pro licensing. They must not publish admin passwords, server credentials, customer data, private license keys, or private deployment logs.
