# Docs And Manuals Updates

docs.talktoai.org is the public documentation hub for the ecosystem.

## Latest Work

- Expanded GitBook-style navigation and product pages.
- Added manual library routes and large downloadable/readable guides.
- Added CallChat, OpenZero, ZeroThink, FreeWebPanel, ZSEC, QuantumEncryption1, business, services, course, FAQ, and contact routes.
- Improved clean URL behaviour and public link structure.
- Reduced internal-sounding wording and made pages speak to visitors.
- Continued UI and speed fixes to reduce scroll glitches and crammed layouts.

## Public Boundary

Docs should explain how products work and where to try them. They must not publish private implementation details, private keys, credentials, raw logs, private customer files, or server-only deployment notes.
