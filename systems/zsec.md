# ZSEC Updates

ZSEC is the security-only Linux update and hardening utility.

## Public Features

- Security update focus.
- Read-only advisory/feed direction.
- SSH lockout guardrail ideas.
- Fail2ban and exposure-check direction.
- Public GitHub documentation.
- PDF/readme/user-guide direction.

## Latest Work

- Repaired public GitHub presentation.
- Improved public page styling and security-focused wording.
- Added clearer install and boundary language.
- Kept ZSEC focused on server security updates and hardening, not remote control.

## Public Boundary

Public ZSEC docs can show safe install and hardening guidance. They should not include private server lists, credentials, logs, or exploit-ready offensive procedures.
