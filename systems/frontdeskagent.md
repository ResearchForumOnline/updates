# FrontDeskAgent Updates

FrontDeskAgent is the self-hosted AI receptionist and customer-response project.

## Public Features

- Website receptionist demo.
- Self-hosted deployment direction.
- OpenZero CPU-first AI lane.
- Optional hosted API fallback where needed.
- Voice assistant direction.
- Business-friendly scripts, FAQs, and setup docs.

## Latest Work

- Improved the public receptionist demo so it can call the OpenZero-backed model route.
- Reduced the old repeated-answer fallback behaviour.
- Added better demo wording for visitor questions.
- Kept abuse controls in place without publishing exact private thresholds.
- Updated GitHub documentation around the live demo and self-hosted use.

## Public Boundary

Public FrontDeskAgent docs should explain what businesses can install and test. They should not include private endpoints, keys, internal paid-service secrets, customer data, or protected operational logs.
